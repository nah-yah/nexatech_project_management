Dossier de gestion de projet pour la migration de l'infrastructure on-premise de NexaCorp vers un cloud hybride (AWS / Azure) et le déploiement du moteur d'IA générative interne **NexaSense**.

## Objectifs du projet

- Migrer 85 % des charges de travail vers le Cloud hybride (AWS / Azure, pilotage FinOps).
- Déployer le module d'IA générative NexaSense sur les directions Support et Finance.
- Réduire les coûts d'exploitation informatique de 30 % et automatiser 45 % des requêtes de niveau 1.
- Mettre le groupe en conformité avec la directive européenne NIS2.
